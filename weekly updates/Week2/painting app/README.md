# Painting App

A small p5.js drawing playground with two ways to create portraits:

- Freehand drawing with brushes.
- Quick character building with shape parts and examples.

## How to use

### 1) Draw with brushes

- Pick a color in **COLOR**.
- Pick a size in **SIZE**.
- Pick a brush in **BRUSH**.
- Draw inside the framed canvas area.

Brush styles:

- **PEN**: smooth ink strokes with tapered width.
- **PENCIL**: textured graphite with grain.
- **HIGHLIGHT**: translucent wide marker.
- **AIRBRUSH**: soft spray.

Shortcuts:

- `1`â€“`5` switch color swatches.
- `E` toggles eraser.
- `U` undo.
- `C` clear.
- `S` save a PNG.
- `Esc` closes open menus.

### 2) Build portraits with shapes

- Use **SHAPES** to choose a part (face, eyes, mouth, etc.).
- Use **SCALE** to change the size of parts.
- Click on the canvas to place parts.

### 3) Try examples

- Click an **EX** button to place a full preset on the canvas.

### 4) Tools

- **ERASE**: toggle eraser for brush strokes and delete parts.
- **LASSO**: select and move a drawn area.
- **UNDO**: step back.
- **CLEAR**: wipe the canvas.
- **SAVE**: export PNG.

## Tips

If you are a talented painter, just use brushes to draw. If not, you can combine shapes to make portraits as you like or get some inspirations from examples.
